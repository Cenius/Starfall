Starfall by ColonelThirtyTwo

Starfall Processor for Garry's Mod + Wiremod

Dependencies

Garry's Mod, of course. For the time being, SF requires Wiremod for the editor and the processor entity. It is planned to add support for non-wiremod setups later, but this isn't a priority at the moment.

Install

Just clone it into your addons folder.
